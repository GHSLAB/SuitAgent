# MCP 配置中心

这个目录包含 SuitAgent 项目中所有 MCP (Model Context Protocol) 服务的配置文件和教程。

## 📁 目录结构

```
.claude/mcp/
├── README.md              # 本文件
├── mineru/                # MinerU 文档转换服务
│   ├── README.md           # MinerU 配置教程
│   ├── OFFICIAL_DOCUMENTATION.md  # 官方完整文档
│   └── install.sh          # MinerU 安装脚本
└── pkulaw/                # 北大法宝法律检索服务
    └── README.md           # 北大法宝配置教程
```

## 🚀 快速开始

1. **查看可用服务**: `ls .claude/mcp/`
2. **配置服务**: 进入对应服务目录查看教程
3. **更新主配置**: 编辑 `.claude/mcp.json` 文件

## 📋 已配置服务

### MinerU - 文档转换
- **目录**: `.claude/mcp/mineru/`
- **功能**: PDF、图片、Word、PPT 等文档转 Markdown
- **特性**: OCR 识别、批量处理、多语言支持
- **输出**: 转换文件保存在原文件同目录，文件名不变，只改扩展名为 `.md`
- **状态**: ✅ 已配置可用
- **官方文档**: [MinerU MCP 官方文档](mineru/OFFICIAL_DOCUMENTATION.md)

### 北大法宝 - 法律检索
- **目录**: `.claude/mcp/pkulaw/`
- **功能**: 法律法规、判例、学术文献检索
- **特性**: 海量法律数据，毫秒级响应，智能语义理解
- **状态**: ✅ 已配置可用

## 🔄 协同使用示例

### 文档分析 + 法律检索
```
用户：请分析这个起诉状 PDF，并查找相关的法律法规
系统：
1. [MinerU] 起诉状.pdf → 起诉状.md（同目录）
2. [北大法宝] 基于起诉内容检索相关法条
3. 生成完整的法律分析报告
```

### 证据材料处理 + 案例检索
```
用户：处理这些证据材料，并查找相似案例
系统：
1. [MinerU] 批量转换证据文档（原文件同目录输出 .md）
2. [北大法宝] 基于证据内容检索相似案例
3. 提供证据评估和案例参考
```
